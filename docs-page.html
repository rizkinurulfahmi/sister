Installasi Kali Linux

"Tugas ini membahas secara komprehensif proses instalasi Kali Linux, dimulai dari langkah awal hingga mencapai titik di mana sistem Kali Linux dapat beroperasi secara optimal. Versi Kali Linux yang digunakan dalam proyek ini adalah 2023.3. Spesifikasi laptop yang dipilih untuk merekam proses instalasi diantaranya:"

  1. Processor Intel Core i3 6006U
  2. RAM 12Gb
  3. SSD 250Gb + HDD 1Tb
  4. Images Kali Linux + VirtualBox + Zoom
